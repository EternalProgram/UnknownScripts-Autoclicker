Thank You For Downloading My Autoclicker, Feel Free To Edit And Publish It, Just Make Sure To Give Me Credit!

I Reccomend It Has A Folder Of Its Own. Then, Press The "Alt" Key To Toggle The Autoclicker.
The Autoclicker Is Off By Default When You Run The Program.
